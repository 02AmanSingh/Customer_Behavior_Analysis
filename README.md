# Customer Behavior Analysis

## 📊 Overview

Customer Behavior Analysis is an end-to-end Data Analytics project focused on understanding customer purchasing patterns, spending behavior, and business performance. The project demonstrates the complete analytics lifecycle, from data cleaning and exploratory data analysis (EDA) to SQL querying, dashboard development, and business reporting.

The objective is to transform raw customer data into actionable insights that help businesses improve customer retention, optimize marketing strategies, and drive revenue growth.

---

## 🎯 Project Objectives

* Analyze customer purchasing behavior and spending patterns.
* Identify high-value customers and key customer segments.
* Discover trends in sales and product performance.
* Generate actionable business insights using data analytics.
* Create an interactive Power BI dashboard for decision-making.

## 🚀 How to Run

### Clone the Repository

```bash
git clone https://github.com/02AmanSingh/Customer_Behavior_Analysis.git
```

### Navigate to the Project Directory

```bash
cd Customer_Behavior_Analysis
```

### Install Required Libraries

```bash
pip install pandas numpy matplotlib seaborn
```

### Run the Analysis

1. Open Jupyter Notebook.
2. Execute the Data Cleaning notebook.
3. Run the EDA notebook.
4. Import the cleaned dataset into MySQL Server.
5. Execute SQL queries for business analysis.
6. Open the Power BI dashboard to explore insights.
